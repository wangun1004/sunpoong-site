FEATURES

주요 기능

⚖️

신고제 / 허가제 자동 판별

연면적·층수·높이 입력만으로 신고제와 허가제를 실시간으로 자동 판별합니다.

🤖

Claude AI 문구 자동 생성

해체 사유, 순서, 비산먼지·소음 저감 계획, 폐기물 처리 계획을 AI가 자동 작성합니다.

📐

CAD 도면 자동 생성

연면적·층수 기반 배치도를 자동 생성하고 DXF(AutoCAD) / SVG 파일로 다운로드합니다.

🏛

5개 구청 맞춤 대응

동구·서구·남구·북구·광산구 각 구청의 요구사항을 반영한 서류를 자동 생성합니다.

🖨️

PDF 출력

브라우저 인쇄 기능으로 즉시 PDF 저장 및 출력이 가능합니다.

🔒

데이터 보안

입력한 모든 데이터는 사용자 기기의 로컬 스토리지에만 저장되며 외부 전송되지 않습니다.

HOW TO USE

사용 방법

건축물 기본 정보 입력

관할 구청, 소재지, 건축물 용도·구조·연면적·층수·높이를 입력하면 신고제/허가제가 자동 판별됩니다.

건축주 및 시공업체 정보 입력

건축주(발주자)와 시공업체 정보, 공사 기간을 입력합니다.

AI 자동 생성

API 키를 설정한 후 AI 자동 생성 버튼을 누르면 해체 계획 문구가 자동으로 작성됩니다.

안전·폐기물 계획 입력

비산먼지·소음 저감 계획, 폐기물 종류별 발생량 예측과 처리 계획을 입력합니다.

도면 업로드 및 CAD 자동 생성

위치도·배치도 등 첨부 도면을 업로드하고, 자동 생성된 CAD 배치도를 다운로드합니다.

PDF 출력 및 구청 제출

최종 확인 후 PDF로 저장하여 관할 구청에 제출합니다. 제출 서류 체크리스트도 자동 제공됩니다.
